# node 升级

### 全局安装n
$ npm install -g n

### 升级到最新稳定版
$ n stable 

### 升级到最新版
$ n latest

### 升级到定制版
$ n v7.10.0

### 切换使用版本
$ n 7.10.0 (ENTER)

### 删除制定版本
$ n rm 7.10.0 

### 用制定的版本执行脚本
$ n use 7.10.0 some.js

### npm 升级到最新版本
//linux下
npm install -g npm

### npm升级到指定版本
//比如升级到5.6.0
npm install -g npm@5.6.0
